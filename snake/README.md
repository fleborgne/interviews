Square Snake Spiral Master
--------------------------
The purpose of this exercise is to display an HTML table with a snake inside.

- The snake is contained in a square (html table),
- Start from a corner and go to the inside by spiral.

Examples:
--------
With Square Edge = 1:

 | 1 |
 | :---: |
 
With Square Edge = 2:

 | 1 | 4 |
 | :---: | :---: |
 | 2 | 3 |
 
With Square Edge = 3:

  | 1     | 8     | 7 |
  | :---: | :---: | :---: |
  | 2     | 9     | 6 |
  | 3     | 4     | 5 |
